# Rice_Grain_Classification
Penelitian ini mengevaluasi enam skenario penanganan class imbalance pada klasifikasi kualitas beras menggunakan MobileNetV3, ResNet50, dan ViT-Small, yaitu Baseline, Weighted Sampling, Weighted Sampling+Augmentation, Proposed Method, Focal Loss-Based, dan Class-Balanced Loss-Based. ResNet50 terbaik dengan akurasi 98,87% dan Macro F1 0,9799.
